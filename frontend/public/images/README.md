# 이미지 폴더 구조

AIM 동아리 웹사이트에서 사용할 이미지들을 저장하는 폴더입니다.

## 폴더별 용도

### `/hero/` - 메인 히어로 섹션
- 메인페이지 상단 히어로 이미지
- 동아리 대표 이미지
- 추천 파일명: `hero-main.jpg`, `hero-bg.jpg`

### `/about/` - 소개 섹션  
- 동아리 소개 관련 이미지
- 활동 사진, 팀 사진 등
- 추천 파일명: `about-team.jpg`, `about-vision.jpg`

### `/activities/` - 활동 섹션
- 동아리 활동 사진들
- 프로젝트, 스터디, 행사 사진
- 추천 파일명: `activity-1.jpg`, `project-showcase.jpg`

### `/members/` - 멤버 관련
- 멤버 프로필 사진
- 팀별 사진
- 추천 파일명: `profile-simjun.jpg`, `team-photo.jpg`

### `/logos/` - 로고 및 아이콘
- AIM 동아리 로고
- 파트너/협력사 로고
- 소셜 미디어 아이콘
- 추천 파일명: `aim-logo.png`, `aim-logo-white.png`

### `/backgrounds/` - 배경 이미지
- 섹션별 배경 이미지
- 패턴, 텍스처 이미지
- 추천 파일명: `bg-gradient.jpg`, `bg-pattern.png`

## 이미지 규격 권장사항

- **히어로 이미지**: 1920x1080px (16:9)
- **활동 사진**: 1200x800px (3:2)  
- **프로필 사진**: 400x400px (1:1)
- **로고**: PNG 형식 (투명 배경)
- **배경**: 고해상도 (1920px 이상)

## 파일 형식
- 사진: `.jpg`, `.jpeg` 
- 로고/아이콘: `.png`, `.svg`
- 최적화된 이미지 사용 권장 (웹 최적화)
